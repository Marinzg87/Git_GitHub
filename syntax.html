Hello to my journey with Git & GitHub

git init - use in the proper folder, and start the Git (version control)

git add file_name.extension - add new file to Git repository

git push origin branch_name - push changes to GitHub repository

git commit file_name.extension -m "comment" - save changes to repository

git status - check if any changes are waiting to commit/push/pull

git pull origin branch_name - pull repo from GitHub

git branch --set-upstream-to=origin/branch_name branch_name - automation, that Git and GitHub will now each other
